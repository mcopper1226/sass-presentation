So, Why Use Sass?

Do more, write less - 120 lines of code versus 380
  1. Efficient - no redundancy!
  2. Well-structured - allows you to organize code into logical groups
  3. Flexible - make changes in one place

Getting Started

Browsers don't understand SASS, so it must be processed into CSS

There are a variety of tools for compiling .scss files - build tools like Gulp/Grunt, Sass Gem, Node-Sass

Features

Feature 1: Variables

A storage container for a CSS value (color, font, numeric value, etc.) that can be used throughout code

							
								$magenta: #d33682;
								$cyan: #35c9be;
								$font_lg: 24px;
								$primary_font: 'Roboto', sans-serif;

								$base_color: $magenta;
								$text_color: $cyan;

								h3 {
									color: $cyan;
									background-color: $base_color;
									font-size: $font-lg;
									font-family: $primary_font;
								}

Feature 2: Nesting

Reduces repetition and allows you to keep everything together

							
								.image-grid {
									width: 80%;
									padding: 1rem;
									img {
										display: block;
										width: 25%;
										height: auto;
									}
									.text-wrapper {
										margin: 1rem 0;
										h2 {
											color: green;
											&:hover {
												color: orange;
											}
										}
										&:after {
											content: 'Hi!';
										}
									}
								}

Feature 3: Partials

Modularize your code and keep everything in functional groups; all partials are compiled into one file so there is only one server call

Feature 4: Operators

Use math operations - addition, division, multiplication - to create more flexible styles

							
								$border_width: 1px;
								$border_width_thicker: $border_width * 5;

								.big-button {
									border: $border_width_thicker solid blue;
								}

Feature 5: Extends

Base a CSS rule on another rule

							
								.button {
									height: 40px;
									line-height: 40px;
									border-radius: 3px;
								}
								.button-solid {
									@extend .button;
									background: green;
									color: yellow;
								}

Feature 6: Mixins

A groups of CSS declarations that can be reused throughout the site

							
						@mixin backImage($image, $position: center, $size: cover) {
						    background-position: $position;
						    background-size: $size;
						    background-repeat: no-repeat;
						    background-image: url($image);
						}

						.home-hero {
							@include backImage('http://unsplash.com/image-1.jpg', center, cover);
						}
